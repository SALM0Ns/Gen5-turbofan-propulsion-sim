# GEN-5 TURBOFAN LAB
### Interactive Thermodynamic Flow & Propulsion Schematic

An advanced, browser-based simulator designed to visualize the **Brayton Cycle** and internal aerodynamics of a 5th-generation low-bypass augmented turbofan engine. This project serves as an interactive educational tool for aerospace propulsion and thermodynamics.

##  Overview
This simulator traces the "Suck, Squeeze, Bang, Blow" journey of air particles through the **AF-119 CORE**. It provides a real-time visualization of how energy is added and extracted within a modern jet engine to achieve both efficient Supercruise and maximum Augmented thrust.

## The 7-Phase Cycle Walkthrough

| Phase | State | Thermodynamic Highlight |
| :--- | :--- | :--- |
| **0** | **System Offline** | Initial structural view of the core and casing geometry. |
| **1** | **Induction** | The 3-stage titanium fan initiates airflow, splitting it between the core and the bypass duct. |
| **2** | **Compression** | 6-stage HPC performs adiabatic compression, spiking pressure to 25.0+ atm. |
| **3** | **Combustion** | Fuel ignition adds massive thermal energy, reaching peak core temps of ~1,900°C. |
| **4** | **Extraction** | HPT and LPT turbines extract work from the gas flow to drive the front compressors. |
| **5** | **Military Thrust** | **Supercruise Enabled.** Maximum dry thrust optimized for Mach 1.5+ efficiency. |
| **6** | **Augmentation** | **Full Afterburner.** Raw fuel is ignited in the exhaust to reach maximum kinetic velocity. |

##  Key Features
* **Real-time Particle Physics:** Dynamic particle velocity and density changes based on thermodynamic states.
* **Live Telemetry Dashboard:** Tracks Core Temperature (T), Pressure (P), and Mach Velocity in real-time.
* **2D Thrust Vectoring Nozzle:** Interactive visualization of mechanical nozzle deflection for high-alpha maneuverability.
* **Thermodynamic Logic:** Accurate representation of Rayleigh losses and work extraction by turbine stages.

##  Tech Stack
* **Engine:** HTML5 Canvas API (Physics-based particle system)
* **Logic:** Vanilla JavaScript (Thermodynamic state machine)
* **UI/UX:** CSS3 Tactical Dashboard Design (High-contrast, responsive layout)

##  Science Highlights
* **Brayton Cycle Visualization:** See the pressure rise in the HPC and the expansion in the nozzle.
* **Supercruise:** Learn how modern core efficiency allows for supersonic flight without afterburners.
* **Bypass Ratio Logic:** Observe the cooling effect and thrust contribution of the bypass air stream.
